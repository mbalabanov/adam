Welcome to the ADAM-API

This application programming interface ( adam-interface.herokuapp.com ) of the Archive of Digital Art and Media. You can access artifacts, persons (artists, curators), teams (artists' collectives), collections, exhibitions and the featured items.

GET (WITHOUT an ID):

  • /artifacts - All artifacts.
  • /persons - All persons.
  • /teams - All teams (artists' collectives).
  • /collections - All collections of artifacts.
  • /exhibitions - All exhibitions.
  • /featured - The three items to be featured on the frontpage.

GET (WITH an ID):

  • /artifacts/id - A specific item from the list of artifacts.
  • /persons/id - A specific item from the list of persons.
  • /teams/id - A specific item from the list of teams (artists' collectives).
  • /collections/id - A specific item from the list of collections of artifacts.
  • /exhibitions/id - A specific item from the list of exhibitions.
  • /featured/id - A specific item of the three featured on the frontpage.

DELETE (WITH an ID):

  • /artifacts/id - Set a specific item from the list of artifacts to unpublished.
  • /persons/id - Set a specific item from the list of persons to unpublished.
  • /teams/id - Set a specific item from the list of teams (artists' collectives) to unpublished.
  • /collections/id - Set a specific item from the list of collections of artifacts to unpublished.
  • /exhibitions/id - Set a specific item from the list of exhibitions to unpublished.

Please note that featured items cannot be deleted/unpublished.

PUT (WITH an ID):

  • /artifacts/id - Edit a specific item from the list of artifacts.
  • /persons/id - Edit a specific item from the list of persons.
  • /teams/id - Edit a specific item from the list of teams (artists' collectives).
  • /collections/id - Edit a specific item from the list of collections of artifacts.
  • /exhibitions/id - Edit a specific item from the list of exhibitions.
  • /featured/id - Edit a specific item from the list of featured on the frontpage.

POST (WITH the parameter 'new'):

  • /artifacts/new - Create a new item in the list of artifacts.
  • /persons/new - Create a new item in the list of persons.
  • /teams/new - Create a new item in the list of teams (artists' collectives).
  • /collections/new - Create a new item in the list of collections of artifacts.
  • /exhibitions/new - Create a new item in the list of exhibitions.

Please note that new featured items should not be created. The total number should be kept at three featured items.
