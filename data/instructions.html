Welcome to the ADAM-API

Here at adam-interface.herokuapp.com you can find the application programming interface of the Archive of Digital Art and Media. You can access artifacts, persons (artists, curators), events, news and the featured items.

This is a project by Marin Balabanov. For more information please visit mbalabanov.github.io/adam/.

GET (root):

Example: GET adam-interface.herokuapp.com/

  • / - These instructions.

GET all:

Example: GET adam-interface.herokuapp.com/all

  • /all - The complete set of data.

GET (WITHOUT an ID):

Example: GET adam-interface.herokuapp.com/artifacts

  • /artifacts - All artifacts.
  • /persons - All persons.
  • /events - All events.
  • /news - All news items.
  • /featured - The four items to be featured on the frontpage.

GET (WITH an ID):

Example: GET adam-interface.herokuapp.com/artifacts/1

  • /artifacts/id - A specific item from the list of artifacts.
  • /persons/id - A specific item from the list of persons.
  • /events/id - A specific item from the list of events.
  • /news/id - A specific news item from the news list.
  • /featured/id - A specific item of the four featured on the frontpage.

DELETE (WITH an ID):

Example: DELETE adam-interface.herokuapp.com/artifacts/1

  • /artifacts/id - Set a specific item from the list of artifacts to unpublished.
  • /persons/id - Set a specific item from the list of persons to unpublished.
  • /events/id - Set a specific item from the list of events to unpublished.
  • /news/id - Set a specific news item to unpublished.

Please note that featured items cannot be deleted/unpublished.

PUT (WITH an ID):

Example: PUT adam-interface.herokuapp.com/persons/0

  • /artifacts/id - Edit a specific item from the list of artifacts.
  • /persons/id - Edit a specific item from the list of persons.
  • /events/id - Edit a specific item from the list of events.
  • /news/id - Edit a specific news item.
  • /featured/id - Edit a specific item from the list of featured on the frontpage.

POST (WITH the parameter 'new'):

Example: POST adam-interface.herokuapp.com/collections/new

  • /artifacts/new - Create a new item in the list of artifacts.
  • /persons/new - Create a new item in the list of persons.
  • /events/new - Create a new item in the list of events.
  • /news/new - Create a new news item.

Please note that new featured items should not be created. The total number should be kept at four featured items.
