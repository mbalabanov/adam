Featured Items

Four Items are featured in the carousel on the frontpage. They can be any of the existing archive item in the categories Artifacts, Persons, and Events. The carousel is limited to four items to stay concise.

Featured items have their own title and description that are independent of the name and desription in the actual archive item's page. The idea is that you can write a promotional text for the featured item. If you want the same text, then simply copy the text from the archive item's page. Please note that the description should not exceed two sentences.

The carousel provides a large canvas to create a great impression. Therefore the images in the carousel are much larger. Please provide the image in a resolution of 1200 x 600 pixels.
